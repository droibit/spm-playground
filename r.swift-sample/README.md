# R.swift sample
Template repository for UIKit sample app generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen).

### Requirements
- Xcode 14+
- [Mint](https://github.com/yonaskolb/Mint#installing) installed.

### Generate sample app

```bash
mint bootstrap
mint run xcodegen xcodegen generate -p ./App
```
